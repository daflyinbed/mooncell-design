# 颜色

Mooncell Design 使用一个类似Radix-ui的颜色系统。其中 强调色(accent)、中性色(gray)、四种语义颜色(info/success/warn/error)，每组颜色都要有4个工具颜色和11个梯度颜色（[accent|gray|info|success|warn|error]-\d）以及和11个梯度颜色对应的11个带透明度的梯度颜色（[accent|gray|info|success|warn|error]-a\d）。下表列出了11个梯度的css变量和对应的用途

| 梯度 | 用途                         |
| ---- | ---------------------------- |
| 1    | 应用背景                     |
| 2    | 柔和背景                     |
| 3    | 组件背景                     |
| 4    | hover时组件背景              |
| 5    | 选中或者按下是的组件背景     |
| 6    | 不可交互组件的边框           |
| 7    | 可交互组件的边框             |
| 8    | 对焦环和更强可交互组件的边框 |
| 9    | 实心背景颜色                 |
| 10   | hover时的实心背景颜色        |
| 11   | 字体颜色                     |

下表列出了4个工具颜色
| 名称 | 用途 |
| ---- | ---------------------------- |
| surface | 表面模式下的背景颜色 |
| indicator | 指示器颜色（checkbox radio选中时的颜色） |
| track | slider progress switch的颜色 |
| contrast | 对比色（比如背景是9时字体的颜色） |

此外还有一组背景颜色。

| 名称                      | 用途                                                  | 默认值                    | 黑暗模式默认值      |
| ------------------------- | ----------------------------------------------------- | ------------------------- | ------------------- |
| --color-background        | 页面背景颜色                                          | #fff                      | gray-1              |
| --color-panel-solid       | 没有语义的组件的背景颜色，比如卡片 表格等组件的背景色 | #fff                      | gray-2              |
| --color-panel-translucent | 同上 但是带透明度                                     | rgba(255, 255, 255, 0.7)  | --gray-a2           |
| --color-panel-surface     | 表单组件的背景颜色                                    | rgba(255, 255, 255, 0.85) | rgba(0, 0, 0, 0.25) |
| --color-overlay           | 弹窗的遮罩颜色                                        | #0006                     | #0009               |
